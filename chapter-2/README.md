# Chapter 2: Routing, Layouts, and Styling

In this chapter, we will build upon the foundation set in Chapter 1. We have a robust testing environment with Vitest, Storybook, and Playwright. Now, it's time to start building the actual application structure.

## Overview

We will cover the following topics:

1. **Routing Basics**: How the file-system based routing works in the Next.js App Router.
2. **Layouts**: Sharing UI across multiple pages.
3. **Styling**: Using Tailwind CSS to build responsive components.
4. **Testing**: Writing tests for our new routes and components.

## Prerequisites

Ensure you have the `chapter-2-start` codebase ready. If you are continuing from Chapter 1, your current state should match.

Run the development server to make sure everything is working:

```bash
cd chapter-2-start
pnpm dev
```

Visit `http://localhost:3000` to see your application.
