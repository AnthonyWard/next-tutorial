# Chapter 1

## Enterprise Network Workaround

Due to authenticated registry requirements within our enterprise network, some of the install commands in the setup instructions may not work as expected.

### Workaround Steps

1. Navigate to the `chapter-1-start` folder
2. Run `pnpm install`
3. Continue from step 3 `03-unit-testing.md` in the setup instructions

This approach bypasses the registry authentication issues and allows you to proceed with the chapter exercises.
